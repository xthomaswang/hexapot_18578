Hexapod Dancing Robot

Problem Description

Team 6 is building a large hexapod robot that can perform choreographed walking and dance-like motions while remaining stable, safe, and modular enough to transport between demos. The project combines locomotion, embedded control, perception, and mechanical integration in a single performance platform.

Our current architecture uses 18 high-torque servos across six 3-DOF legs, an ESP32 for low-level real-time control, and a Raspberry Pi 5 for gait generation, calibration, and camera-based computer vision. The robot also includes an IMU, wireless control path, emergency stop, and AprilTag-ready camera stack to support safe manual operation and future autonomous alignment tasks.

Key Features

18 Servo Joints

Six legs with coxa, femur, and tibia joints provide the maneuverability needed for tripod gait and expressive motion.

Split Compute Architecture

The Raspberry Pi handles planning and perception while the ESP32 handles timing-sensitive PWM output and control logic.

Physics-Backed Development

MuJoCo simulation lets the team tune geometry, gait timing, and stability before committing changes to hardware.
