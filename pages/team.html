Thomas Wang

Software & System Integration

Software architecture and communication protocol design between the Raspberry Pi and ESP32 over UART. Developed the layered Raspberry Pi control software: inverse kinematics engine, motion planning (gait generation), AprilTag-based calibration, and manual/automated control modes.

Leena Aljurashi

Embedded Systems

ESP32 embedded firmware using FreeRTOS with six concurrent real-time tasks: Bluetooth communication, IMU reading, vision data reception, main control loop, servo output driving, and safety monitoring. Also responsible for controller firmware and Bluetooth link.
